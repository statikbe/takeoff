BEM

Takeoff uses a component-based CSS class system called BEM (Block, Element, Modifier).

As an example, let's take a look at this commonly used card component:

Card title

Lorem ipsum dolor sit amet, consectetur.

Block

A block is a standalone entity that is meaningful on its own.

In the card example, card is a block class.

Element

An element is a part of a block, that has no meaning outside of the block context.

In the card example, card__title and card__text are element classes.

Modifier

A modifier class changes the appearance of the default block.

In the card example, card--link is a modifier class.

Rhythm

To give the web application a clean and uniform look, we use predefined rhythm variables. They are based off the line-height of the body text (which is usually 24px). Throughout our components, these variables are commonly used for setting margins and padding. There are a total of 6 rhythm variables:

xxs (6px)
xsm (12px)
sml (24px)
med (36px)
lrg (48px)
xlr (60px)

See also: Spacing

Repeating grid

In a repeating grid, every grid item has the same width. The parent element class decides the layout:

grid--<n>-col@<breakpoint>

<breakpoint> refers to the minimum viewport width for this class to apply. See breakpoints.
<n> refers to the number of colums.

Child elements must always have the grid__item class, and preferably no other classes.

Let's create a repeating grid that starts out as a single column, and scales up to 3 columns on larger screens. Try resizing your screen to see the result!

Custom grid

In a custom grid, grid items can have different widths. This is mostly used for creating layouts. The child element class decides the layout:

grid__<n>@<breakpoint>

<breakpoint> refers to the minimum viewport width for this class to apply. See breakpoints.
<n> refers to the number of colums this child spans. We use a 12 column grid system.
Note: do not forget the grid__12 class.

Parent elements must always have the grid class, and preferably no other classes. (except for the spacer class)
